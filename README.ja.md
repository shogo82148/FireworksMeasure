FireworksMeasure

======================
花火までの距離を測定するためのAndroidアプリです。

## 使い方
1. FireworksMeasureを起動すると、カメラプレビューが表示されます。
2. 花火が見えたら画面をタップしましょう。
3. 花火が画面中央に写るよう、端末を動かします。
4. 花火の爆発音が聞こえたら、もう一度画面をタップします。
5. すると、花火までの距離がわかります！
